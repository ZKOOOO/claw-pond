OpenClaw 平台

把 OpenClaw 资源池、任务单和个人龙虾放进一个中文控制台

这个页面直接连接当前 Spring Boot 服务。你可以完成注册登录、管理自己接入的 OpenClaw、 在共享 OpenClaw 池里按标签筛选资源、创建任务单干活，也可以上传自己的龙虾并为它们贴上擅长标签。

一步 登录平台

注册或登录后，JWT 会被自动保存在浏览器本地，页面刷新后也能继续使用。

二步 管理 OpenClaw

接入自己的 OpenClaw，挂上标签，进入共享池后即可被任务选择。

三步 上传龙虾干活

把自己的龙虾文件上传到平台，贴上擅长标签，形成可复用的个人资产池。

操作指引 推荐的完整流程
  1. 注册并登录账号，确认“当前会话”里已经显示你的身份信息。
  2. 在“我的 OpenClaw 管理”中录入实例地址、标签和说明。
  3. 进入“OpenClaw 资源池”，按标签筛选想要干活的实例。
  4. 选中实例后，在“任务单”里填写任务标题、说明和期望标签。
  5. 如果你有自己的龙虾文件，可以在“我的龙虾”里上传并打标签沉淀资产。
提示

标签输入支持中英文逗号分隔，例如：`推理, OCR, 高并发`。

操作参考

核心接口一览

用户 /api/auth/register / /api/auth/login

用户注册、登录和会话获取。

管理 /api/openclaws

创建、更新、删除自己管理的 OpenClaw 实例。

资源池 /api/openclaw-pool

按标签筛选启用中的 OpenClaw 共享池。

任务单 /api/work-jobs

选择 OpenClaw 后创建任务，并查看自己的任务列表。

龙虾 /api/lobsters

上传、查看和下载自己的龙虾文件资产。
