OpenClaw 平台

把资源池、任务单和龙虾资产统一到一个中文控制台

你可以在这里完成用户登录注册、管理自己的 OpenClaw、进入共享资源池按标签筛选实例、 创建任务单并绑定自己的龙虾文件，还能在管理员身份下看到平台总览。

步骤一 登录平台

完成注册或登录后，JWT 会自动保存在浏览器本地。

步骤二 配置 OpenClaw

给实例打标签，实例启用后会自动进入共享 OpenClaw 池。

步骤三 带着龙虾干活

上传龙虾、选择 OpenClaw、创建任务单，形成完整工作链路。

操作指引 推荐流程
  1. 先注册或登录账号，确认当前会话已生效。
  2. 录入自己的 OpenClaw，并为实例设置标签和状态。
  3. 在共享 OpenClaw 池按标签筛选实例，选择一个可用节点。
  4. 如果你有自己的龙虾，先上传龙虾文件并贴上擅长标签。
  5. 创建任务单时选择 OpenClaw，并可附带自己的龙虾一起干活。
提示

标签输入支持中英文逗号分隔，例如：`推理, OCR, 高并发`。

操作参考

核心接口一览

用户 /api/auth/register / /api/auth/login

用户注册、登录和会话获取。

管理 /api/openclaws

创建、更新、删除自己管理的 OpenClaw 实例。

资源池 /api/openclaw-pool

按标签筛选启用中的 OpenClaw 共享池。

任务单 /api/work-jobs

选择 OpenClaw，可附带龙虾，再创建任务单。

龙虾 /api/lobsters

上传、查看和下载自己的龙虾文件资产。

管理员 /api/admin/overview

查看平台用户、实例、任务单、龙虾和标签总览。
