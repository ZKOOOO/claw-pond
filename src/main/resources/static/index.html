OpenClaw Platform

Bring scattered OpenClaw nodes into one control room

This page talks directly to the current Spring Boot service. You can register, sign in, inspect the active session, onboard external OpenClaw instances, and review the inventory owned by the current user.

Step 1 Create an account

Registration returns a JWT immediately and the page stores it locally.

Step 2 Onboard OpenClaw

Submit a name, base URL, and external ID to bind an instance to your account.

Step 3 Review inventory

The inventory view shows ownership, endpoint address, and creation time.

Operator Guide Fastest path through the UI
  1. Register a new account or sign in with an existing one.
  2. Check that the Session card shows your username and role.
  3. Fill in the OpenClaw public URL and a globally unique external ID.
  4. Create the instance and refresh the inventory list to verify it was stored.
Note

`baseUrl` must start with `http://` or `https://`, and `externalId` must be unique.

How To Operate

API Flow At A Glance

POST /api/auth/register

Create a user account and receive a JWT.

POST /api/auth/login

Authenticate with email and password, then receive a JWT.

GET /api/auth/me

Inspect the current authenticated identity.

POST /api/openclaws

Create a managed record for an external OpenClaw node.

GET /api/openclaws

Load the current user's visible instance list.
